# Description

Simple ToDo App
You can:

- Add a new tasks
- Delete your tasks
- Complete your tasks
- You can set your date for task
- You can also set his priority

# Technologies i used

- HTML
- CSS
- Java Script
- Vite
- React.js
  - Components
  - Hooks useState()
  - Conditional rendering
  - Props
  - JSX
